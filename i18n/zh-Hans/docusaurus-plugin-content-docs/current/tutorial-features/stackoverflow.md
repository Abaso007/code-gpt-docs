# CodeGPT 上的 Stack Overflow

1. **搜索**: 立即在 Stack Overflow 上搜索与您的问题相关的内容。
2. **总结**: CodeGPT 将阅读帖子并生成简明摘要。
3. **回应**: 结合收集的信息全面回答您的问题。

## 如何使用:

:::caution 仅适用于部分来自 Marketplace 的代理
- 确保您的 CodeGPT 账户已同步。
- 选择一个模型或代理。
- 添加您想用作 Stack Overflow 搜索上下文的数据或文件。
- 输入 `/` 并选择 `/Stack Overflow` 工具，然后写下您的问题。
- 要验证链接，只需点击它。

:::
