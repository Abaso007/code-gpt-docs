# 单元测试

单元测试 CodeGPT 功能使您能够快速生成代码的单元测试。选择特定的代码段可以提示 CodeGPT 创建相应的单元测试，从而提高代码的可靠性和质量。

## 如何使用：
:::note CodeGPT: 单元测试 CodeGPT
要使用此功能，请使用您的 CodeGPT 帐户登录，您每天将有有限的免费互动次数。您可以订阅我们的计划之一，以获得更多的互动次数，并访问选定的 AI 模型，而无需担心为每个提供商创建 API 密钥或帐户（我们让您轻松！）。如果您想免费使用 CodeGPT，您必须添加所需提供商（例如，Anthropic、OpenAI）的自己的 API 密钥，并且您将直接向他们支付使用费用。
:::

- 选择代码：突出显示代码片段
- 输入 `/` 斜杠，选择 `单元测试`，按 Enter 键发送请求
