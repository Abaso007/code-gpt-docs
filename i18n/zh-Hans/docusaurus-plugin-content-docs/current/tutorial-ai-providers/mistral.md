## 连接您的 Mistral 账户
- 在 [Mistral](https://auth.mistral.ai/ui/login) 创建一个账户
- 转到您的 [平台 / API 密钥](https://console.mistral.ai/users/api-keys/)
- 创建并复制 API 密钥
- 在 VSCode 中按下 ```cmd + shift + p``` 并搜索 **`Set API Key`**
- 粘贴您的 API 密钥

:::caution 移除密钥
如果您想从 CodeGPT 中移除 API 密钥，请打开命令面板并搜索 `CodeGPT: Remove API KEY`。
:::
