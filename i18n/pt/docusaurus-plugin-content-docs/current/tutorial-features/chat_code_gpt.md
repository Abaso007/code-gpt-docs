# CodeGPT Chat

O recurso CodeGPT Chat melhora sua experiência de desenvolvimento, permitindo que você se envolva em conversas com Modelos de Linguagem de Grande Escala (LLM) diretamente no seu editor de texto. Este tutorial irá guiá-lo através dos passos para aproveitar ao máximo esta poderosa ferramenta.

## Como Usar:
- Abra a Aba de Chat
- Navegue até a aba de ferramentas e clique em `Chat`. Por padrão, `Claude-3.5-Sonnet` responderá no chat.
- Uma vez que a aba de chat esteja aberta, você pode iniciar uma conversa digitando uma mensagem e pressionando `Enter`.
- Se você solicitar código, copie ou cole diretamente no arquivo com o botão `Inserir código`.
- Você pode limpar o chat clicando no botão `Novo chat` ou `Recarregar` a pergunta.

## Usando Seleção de Código:
Selecione o código desejado no seu editor por nível para compartilhar trechos de código. O código selecionado será automaticamente inserido na interface de chat, permitindo que o CodeGPT forneça respostas mais precisas e específicas ao contexto. Chame a função usando `@`.

:::note Seleção de Código

Para ler o nome do arquivo ou segmento, você pode usar a barra deslizante para navegar pelo nome.

### Nível de Contexto

Por favor, habilite-o, e a IA será capaz de entender o contexto do seu código.

Você tem três níveis de contexto:

- `Código Selecionado`: código selecionado manualmente
- `Base de Código`: selecionamos automaticamente a base de código. Por favor, indexe a base de código antes no botão `Base de Código`
- `Insights Profundos`: enriquecemos as respostas com gráficos de conhecimento de código.

2. Escolha `#Arquivos` para adicionar um arquivo do seu projeto atual como contexto

## Atalhos
Você pode usar os atalhos para aproveitar ao máximo o recurso CodeGPT Chat. Aqui estão alguns dos atalhos mais úteis:

- [`/Fix`](https://docs.codegpt.co/docs/tutorial-features/find_problems): Corrigir o código selecionado
- [`/Explain`](https://docs.codegpt.co/docs/tutorial-features/code_explanation): Explicar o código selecionado
- [`/Refactor`](https://docs.codegpt.co/docs/tutorial-features/code_refactoring): Refatorar o código selecionado
- [`/Document`](https://docs.codegpt.co/docs/tutorial-features/code_documentation): Gerar documentação para o código selecionado
- [`/Inline code`](https://docs.codegpt.co/docs/tutorial-features/code_inline): editar palavras específicas dentro do código selecionado.
- [`/Unit Test`](https://docs.codegpt.co/docs/tutorial-features/unit_testing): Testar o código selecionado
- [`/Comment`](ddd): Comentar o código selecionado
- [`/StackOverflow`](https://docs.codegpt.co/docs/tutorial-features/stackoverflow): Pesquisar o código selecionado no StackOverflow
- [`/Code Builder`](https://docs.codegpt.co/docs/tutorial-features/code_builder): Gerar um construtor de código para o código selecionado
- [`/Code Interpreter`](https://docs.codegpt.co/docs/tutorial-features/code_interpreter): Interpretar o código selecionado sem executá-lo
- [`/Planning`](dddd): permitir que agentes especialistas do nosso Marketplace ajudem você a planejar o código.
